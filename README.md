Intro                 
=====                 
  
Alfred 2 workflow: 快速切换 DNS 服务器设置
  
预设的 DNS 配置包括:  
  
* Default DNS 
  * empty  
* Alibaba Public DNS 
  * 223.5.5.5 
  * 223.6.6.6                                                                                                 
* V2EX Public DNS
  * 199.91.73.222 
  * 178.79.131.110
* 114 Public DNS
  * 114.114.114.114 
  * 114.114.115.115
* Google Public DNS 
  * 8.8.8.8 
  * 8.8.4.4
* OpenerDNS 
  * 42.120.21.30
  
Usage                 
=====                 

Alfred 框中输入 sdns, 从下拉列表中选取需要的 DNS 后，回车即可
